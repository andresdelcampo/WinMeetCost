# WinMeetCost
A transparent meeting cost tracker running in the bottom right corner of the screen. 
It allows adding and removing attendees as the meeting continues by clicking the + and -.

Fully configurable for currency symbol, average hourly rate, default number of attendees and screen location. 
Edit WinMeetCost.dll.config to customize it. 

Fun quick project made in WinForms (Visual Studio 2022). Compatible with Windows and .NET 6.
